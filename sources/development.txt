Development & Contribution
==========================

Install from Github:

    $ cd workspace
    $ git clone <Your fork Url>


Build Documents:

    $ cd path/to/the/project
    $ cd sphinx
    $ make html

or

    $ ./watch


Start the document http server locally:

    $ gem install jekyll
    $ cd path/to/the/project.io
    $ jekyll serve -w

or

    $ ./watch
