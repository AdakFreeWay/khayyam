Installation
============

From PyPI
---------

pip::

  $ pip install khayyam

Specific version::

  $ pip install "khayyam>=2.0.0"

easy_install::

  $ easy_install khayyam


From Source
-----------
::

  $ cd path/to/source/directory
  $ pip install .

or::

  $ python setup.py install

Development, editable::

  $ cd path/to/source/directory
  $ pip install -e .



From Github
-----------
Latest development code::

  $ pip install git+https://github.com/pylover/khayyam.git
